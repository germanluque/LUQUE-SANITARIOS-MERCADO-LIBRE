# Sanitarios Luque | Catálogo Premium

Este proyecto es una plataforma web moderna y minimalista diseñada para **Sanitarios Luque**. El objetivo es presentar un catálogo de productos de alta gama con una estética limpia, profesional y atractiva.

## 🚀 Tecnologías Utilizadas
- **React 19**
- **Vite** (Build tool ultra rápido)
- **Framer Motion** (Animaciones fluidas y transiciones)
- **Lucide React** (Iconografía minimalista)
- **Vanilla CSS** (Diseño personalizado y ligero)

## 🎨 Diseño
- **Color Predominante**: Naranja Luque (`#FF7A00`)
- **Estética**: Minimalismo, lujo y claridad visual.
- **Responsive**: Totalmente adaptado para móviles y tablets.

## 📦 Estructura del Proyecto
- `src/App.tsx`: Componente principal con la lógica del catálogo y secciones.
- `src/index.css`: Sistema de diseño y variables globales.
- `public/`: Contiene los recursos visuales del catálogo (imágenes generadas por IA).

## 🛠️ Desarrollo
Para ejecutar el proyecto localmente:
```bash
npm install
npm run dev
```

Para generar la versión de producción:
```bash
npm run build
```
